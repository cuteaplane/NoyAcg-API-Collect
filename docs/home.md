# 获取主页数据 (Home)

## 简述

该接口用于获取 **NoyAcg 首页聚合数据**。
返回内容包含：

* 顶部公告（貌似现在已被在ui中隐藏）
* 漫画推荐榜单
* 流媒体推荐
* 推荐标签

榜单按照不同维度划分，例如：

* 日收藏 (`favDay`)
* 日阅读 (`readDay`)


接口支持通过 **请求头控制 R18 内容过滤**，见`allow-adult` 说明。

---

# 接口信息

| 项目           | 内容                                     |
| ------------ | -------------------------------------- |
| Endpoint     | `https://beta.noyteam.online/api/home` |
| 请求方式         | `POST`                                 |
| Content-Type | `application/x-www-form-urlencoded`    |
| 返回格式         | `application/json`                     |
| 鉴权方式         | Cookie (`NOY_SESSION`)                 |

---

# 请求头 (Headers)

| 字段名           | 值                      | 说明     |
| ------------- | ------------------------- | ------ |
| `Cookie`      | `NOY_SESSION=xxxx`           | 用户会话   |
| `allow-adult` | `true` / `false` / `both`   | 内容分级 |

### allow-adult 说明

| 值       | 含义         |
| ------- | ---------- |
| `true`  | 仅返回 R18 内容 |
| `false` | 仅返回全年龄内容   |
| `both`  | 返回所有内容     |

---


# 返回数据结构

返回 JSON 顶层结构如下：

```json
{
  "bulletin": {},
  "favDay": [],
  "readDay": [],
  "fs": [],
  "proportion": [],
  "stream": [],
  "tags": [],
  "status": "ok"
}
```

---

# 返回字段说明

## 1. bulletin（首页公告）

| 字段     | 类型      | 说明   |
| ------ | ------- | ---- |
| `text` | string  | 公告内容 |
| `show` | boolean | 是否显示 |

示例：

```json
{
  "text": "點擊廣告，救救Noy！",
  "show": true
}
```

---

# 2. favDay（今日收藏榜）

类型：`Book[]`

# 3. readDay（今日阅读榜）

类型：`Book[]`

# 4. fs（推荐）


类型：`Book[]`

# 5. proportion（高质漫画）

系统推荐内容。

类型：`Book[]`

---

# Book 对象结构

用于表示漫画/书籍。

| 字段             | 类型     | 说明       | 示例             |
| -------------- | ------ | -------- | -------------- |
| `Bid`          | int    | 书籍ID     | `15969`        |
| `Mode`         | int    | 内容类型     | `1`            |
| `Bookname`     | string | 书名       | `"敗北女角太多了！"`   |
| `Description`  | string | 简介       | `""`           |
| `Author`       | string | 作者       | `"いたち 雨森たきび"`  |
| `Pname`        | string | 主要角色     | `"綾小路清隆"`      |
| `Ptag`         | string | 标签（空格分隔） | `"戀愛 校園 輕小說"`  |
| `Otag`         | string | 来源类型     | `"原創"`         |
| `Time`         | int    | 更新时间戳    | `1772638599`   |
| `Views`        | int    | 浏览量      | `10522`        |
| `Favorites`    | int    | 收藏数      | `1170`         |
| `F`            | bool   | 当前用户是否收藏 | `false`        |
| `Len`          | int    | 内容长度类型   | `1 / 2`        |
| `Adult`        | int    | 是否R18    | `0 / 1`        |
| `Status`       | int    | 状态       | `0=连载`, `1=完结` |
| `publish_year` | string | 发布年份     | `"2024"`       |
| `RatingSUM`    | float  | 平均评分     | `9.8`          |
| `RatingCount`  | int    | 评分人数     | `65`           |
| `RatingBayes`  | float  | 贝叶斯评分    | `0`            |

---

# 6. stream（流媒体）

首页推荐的 **动画/番剧内容**。

类型：`Stream[]`

---

# Stream 对象结构

| 字段          | 类型     | 说明       | 示例               |
| ----------- | ------ | -------- | ---------------- |
| `sid`       | int    | 剧集ID     | `75`             |
| `name`      | string | 剧名       | `"咒術迴戰 死滅迴游 前篇"` |
| `des`       | string | 剧情简介     | `"經歷了澀谷事變..."`   |
| `tags`      | string | 标签（逗号分隔） | `"動作,冒險,奇幻"`     |
| `pages`     | string | 可观看集数列表  | `"1,2,3,4,5"`    |
| `type`      | int    | 类型标识     | `0`              |
| `count`     | int    | 总集数      | `9`              |
| `time`      | int    | 创建时间     | `1768034642`     |
| `up_time`   | int    | 最后更新时间   | `1772806735`     |
| `f`         | bool   | 是否收藏     | `false`          |
| `views`     | int    | 播放量      | `989`            |
| `favorites` | int    | 收藏数      | `41`             |

---

# 7. tags（标签列表）

首页可用标签。

类型：

```
string[]
```

示例：

```json
[
  "冒險",
  "奇幻",
  "科幻",
  "懸疑",
  "校園",
  "戀愛",
  "恐怖",
  "日常",
  "治癒"
]
```

---

# status

| 值       | 含义   |
| ------- | ---- |
| `ok`    | 请求成功 |
| `error` | 请求失败 |

---

# 请求示例

### curl

```bash
curl https://beta.noyteam.online/api/home \
-X POST \
-H "Cookie: NOY_SESSION=xxxx" \
-H "allow-adult: "false" \
```
